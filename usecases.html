Cross-Industry Patterns

Before diving into specific industries, here are the five agent archetypes you'll see across every vertical. Each maps to a specific set of Okta capabilities.

Archetype 1

Customer-Facing Agents

Agents that interact with end customers.

Primary: Token Vault + CIBA

Industries: FinServ, Retail, Insurance, Healthcare

Risk: Agent accesses all customer data regardless of authorization.

Archetype 2

Internal Knowledge Agents

Search and synthesize internal docs.

Primary: FGA + RAG Authorization

Industries: Technology, Government, Manufacturing

Risk: Confidential docs leak into any employee's LLM context.

Archetype 3

Compliance & Audit Agents

Monitor regulations, generate reports.

Primary: OIG + System Logs + ISPM

Industries: FinServ, Healthcare, Insurance, Gov

Risk: Compliance agent becomes ungoverned access point.

Archetype 4

Multi-System Orchestration

Coordinate across SaaS + internal APIs.

Primary: XAA + MCP + Token Vault

Industries: Technology, Manufacturing, Retail

Risk: Single compromise cascades across all systems.

Archetype 5

Human-Approved Workflows

High-value actions requiring sign-off.

Primary: CIBA + Delegation Chain

Industries: FinServ, Healthcare, Insurance, Gov

Risk: Agent auto-approves wire transfers, medication changes, etc.
